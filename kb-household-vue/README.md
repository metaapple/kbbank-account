# KB Household Vue

Figma Site 원본 코드를 직접 가져올 수 없는 환경에서, 공개 사이트 제목 **"국민은행 가계부 서비스"** 를 기준으로 Vue 3 + Vite 구조로 재구성한 예제 프로젝트입니다.

## 포함 화면
- 홈
- 로그인
- 회원가입
- 가계부 대시보드

## 실행 방법
```bash
npm install
npm run dev
```

## 빌드
```bash
npm run build
```

## 라우트
- `/`
- `/login`
- `/signup`
- `/ledger`

## 참고
배포된 Figma Site는 원본 `.fig` 파일이나 소스 코드를 직접 다운로드할 수 없어서, 완전히 동일한 복제본이 아니라 **디자인 톤과 서비스 성격을 반영한 재구성본** 입니다.
